# 162 Gallery — Implementation Plan

## Design Summary

**Vibe:** Renaissance-meets-modern. Michelangelo's Creation of Adam as the hero motif. Fresco plaster texture throughout. Poster serif typography. Subtle CSS grid overlay for the modern touch. Muted earthy palette.

**Color Palette:**
- Background: `#D6CFC6` (sampled from tileable-bg.png — warm grey-stone)
- Surface/card: `#E8E2DA` (slightly lighter)
- Primary text: `#1C1C1A`
- Accent: `#C4603E` (terracotta)
- Subtle accent: `#6B7C5E` (sage green)
- Grid overlay: `#1C1C1A` at 6% opacity

**Typography:**
- Display/headings: `Cormorant Garamond` (Google Fonts) — Renaissance serif, free
- Body: `Inter` (Google Fonts) — clean modern sans, contrast with display

**Assets (all in `/assets/`):**
- `adam-hand.png` — left hand, reaches right, transparent bg
- `god-hand.png` — right hand, index finger points left, transparent bg
- `tileable-bg.png` — fresco plaster texture, seamless

## Data Flow

### `repos.json` (manually maintained)
```json
[
  { "repo": "UCLA-MAEHS-162/team-nova", "year": 2026 },
  { "repo": "UCLA-MAEHS-162/team-atlas", "year": 2026 }
]
```
You edit this once per cohort. 16 entries for 2026.

### `manifest.json` (in each student repo root)
```json
{
  "title": "Project Nova",
  "authors": ["Alice Chen", "Bob Kim", "Carol Liu"],
  "thumbnail": "assets/thumbnail.png",
  "keywords": ["distributed systems", "consensus", "Raft"],
  "summary": "One paragraph. Plain text, no markdown.",
  "demo_url": "https://...",
  "video_url": "https://youtube.com/..."
}
```

### `scripts/fetch-projects.mjs`
Runs at build time (GitHub Actions). For each repo in `repos.json`:
1. Fetch repo metadata from GitHub API (stars, contributors, last updated)
2. Fetch `manifest.json` from the repo's default branch
3. Fetch README (base64, store raw markdown string)
4. Write combined result to `public/data/projects.json`

### `public/data/projects.json` (auto-generated, committed by CI)
```json
[
  {
```

## Hero Animation Detail

**Layout:** Three elements in a flex row centered in the viewport
```
[adam-hand]  [title block]  [god-hand]
```

**Framer Motion sequence:**
- On mount: adam-hand starts at `x: -200, opacity: 0`, god-hand at `x: 200, opacity: 0`
- Animate to `x: 0, opacity: 1` with `duration: 1.8, ease: "easeOut"`
- After settle: continuous subtle `x` oscillation ±6px, `duration: 3, repeat: Infinity, repeatType: "mirror"` — gives the feeling they're still slowly reaching
- Title fades in after hands settle (`delay: 1.6`)

**CSS Grid Overlay (`GridOverlay.tsx`):**
- `position: absolute, inset: 0, pointerEvents: none`
- CSS `repeating-linear-gradient` in both axes at 60px intervals
- Color: `rgba(28, 28, 26, 0.06)`
- Applied to the full hero section, and optionally a very faint version site-wide

## GitHub Actions Workflows

### `fetch-data.yml`
```yaml
on:
  schedule:
    - cron: '0 6 * * *'   # nightly 6am UTC
  workflow_dispatch:        # manual trigger
```
Steps: checkout → setup Node → run `scripts/fetch-projects.mjs` (uses `GITHUB_TOKEN`) → commit `public/data/projects.json` if changed → push

### `deploy.yml`
```yaml
on:
  push:
    branches: [main]
```
Steps: checkout → setup Node → `npm ci` → `npm run build` → deploy `dist/` to `gh-pages` branch via `peaceiris/actions-gh-pages`

## Build Order

1. **Project scaffold** — `npm create vite`, install deps, configure Tailwind + shadcn/ui
2. **Global styles** — font imports, CSS variables, background texture, grid overlay
3. **Hero component** — layout + Framer Motion animation
4. **Static home page** — About section + placeholder card grid
5. **`repos.json` + `fetch-projects.mjs`** — data pipeline script
6. **`ProjectCard` + `ProjectGrid`** — wire up to `projects.json`
7. **`Project` detail page** — README renderer
8. **GitHub Actions** — fetch-data + deploy workflows
9. **`CONTRIBUTING.md`** — manifest.json instructions for students
10. **Polish** — responsive layout, hover states, accessibility
